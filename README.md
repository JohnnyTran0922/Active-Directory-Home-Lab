# Active-Directory-Home-Lab
Hands-on Active Directory home lab built in VMware — OUs, users, groups, GPOs, and account management for Tier 1 helpdesk practice.

## What This Lab Covers
- Deployed AD DS and DNS roles on Windows Server 2019
- Created Organizational Units (OUs) mirroring a company structure
- Created and managed domain user accounts
- Created security groups and assigned user memberships
- Performed password resets, account unlocks, and account disabling
- Created and configured a Password Policy GPO with account lockout settings

## Environment
- Hypervisor: VMware Workstation Pro
- Server OS: Windows Server 2019
- Domain: lab.local
- Tools: Active Directory Users and Computers, Group Policy Management Console
